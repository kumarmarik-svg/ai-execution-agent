# AI Execution Intelligence Agent

This project is an AI-powered task execution planner built using Streamlit.

## 🚀 Features

* Converts user instructions into actionable tasks
* Assigns tasks based on team skills and availability
* Maintains task dependencies
* Ensures only execution-focused tasks (no planning or irrelevant outputs)
* Handles vague and detailed inputs

## 🧠 Approach

The system uses an LLM (via OpenRouter free model) with controlled prompts and filtering logic to generate stable and relevant task outputs.

## ⚠️ Limitations

* Since a free LLM model is used, responses may sometimes be inconsistent
* The system is still under improvement for better accuracy and stability

## 🔧 Tech Stack

* Python
* Streamlit
* OpenRouter (LLM API)

## 📌 Note

This is a prototype focused on demonstrating AI-driven task planning and execution logic.
